# Caja Diaria — Documentación

Aplicación personal de registro de gastos y proyección financiera.
Stack: Laravel 13 · Inertia 3 · Vue 3 · TypeScript · Tailwind · shadcn-vue.

## Índice

| Documento                       | Descripción                                                |
| ------------------------------- | ---------------------------------------------------------- |
| `analisis-sistema-actual.md`    | Cómo funciona hoy la planilla `proyeccion-2026.xlsx`       |
| `plan-de-trabajo.md`            | Plan de trabajo por fases: arquitectura, modelo, tareas    |

## Cómo usar este plan

1. Lee primero `analisis-sistema-actual.md` para entender qué debe replicar la app.
2. Sigue `plan-de-trabajo.md` fase por fase (§5). Cada fase es entregable por
   separado y deja la app funcional hasta ese punto.
3. Resuelve las preguntas abiertas del §10 del plan antes de arrancar la Fase 1.
